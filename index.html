Les commandes
  • Git init: initialise le dossier "Je commence mon projet"
  • Mkdir: création d'un nouveau directory
  • Touch:création d'un nouveau fichier
  • Git Status: Vérifie l'état du projet/des fichiers
  • Git Add: Ajouter les nouveaux fichiers et dossier.
  • Git commit -m"le nom du commit": Enregistrement des fichiers
  • Git push -u origin master: Envoie des fichiers du Home vers les github repository
  • Git pull origin master: Récuperer les fichiers/modifs de github vers le home
  • Git log: affiche les anciens commit
  • Git log --oneline: Affiche les commit sur une ligne
  • Git log --oneline -p (nom du fichier): Affiche les commit du fichier nommé
  • Git diff: Affiche les dernières modifications
  • Git config global color-uni:< Ajout des couleurs
  • Git config --list: Affiche l'état de la config
  • Git checkout "ID" "nom du fichier": Permet de revenir a une version antérieure
  • Git branch: Affiche les branch
  • Git branch checkout "nom de la branch": Permet de passer a la branch nommée
